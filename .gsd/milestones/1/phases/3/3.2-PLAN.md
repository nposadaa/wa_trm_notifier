# Plan 3.2: Playwright Forwarding Automation

## Objective
Establish a robust, automated forwarding mechanism for WhatsApp Web using **Python + Playwright**. This pivot avoids official API group messaging limitations by simulating human-like forwarding from your primary account to a predefined list of recipients.

## User Review Required

> [!CAUTION]
> **Account Ban Risk**: Building custom automation for WhatsApp Web violates WhatsApp's Terms of Service. However, since we simply simulate a few UI clicks per day, the risk is minimized.
